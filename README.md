## Instructions to Build
Installation Requirements, all located in CarePackage if needed
-Apache Ant
-Java

Make sure ant has a path in the systems enviroment variables named: ANT_HOME
likewise, can also add in a path to the java jdk named: JAVA_HOME

from there, all you need is the build.xml file to build a fresh copy of the project
